# TestApi

A webapi project created with Nexo CLI.

## Getting Started

### Prerequisites

- .NET 8.0 SDK or later
- Your preferred IDE (Visual Studio, VS Code, Rider, etc.)

### Running the Project

```bash
cd TestApi
dotnet run
```

### Building the Project

```bash
dotnet build
```

### Running Tests

```bash
dotnet test
```

## Project Structure

This project follows standard .NET conventions:

- `src/` - Source code
- `tests/` - Test projects
- `docs/` - Documentation
